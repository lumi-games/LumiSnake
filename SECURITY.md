## Reporting a Vulnerability

We take the security of our project seriously. If you have found a vulnerability, please report it to us as quickly as possible.

### How to report

Please report vulnerabilities by emailing **chatgpt11320@gmail.com** or by using the **"Private vulnerability reporting"** feature in the Security tab of this repository.

*   Please do not report security vulnerabilities through public GitHub issues.
*   We aim to respond to your report within 48 hours.
*   We will keep you updated on our progress as we investigate and fix the issue.

Thank you for improving the security of our project!
